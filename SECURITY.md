# Security Policy

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

- Use GitHub's [private vulnerability reporting](https://github.com/meno-so/meno-astro/security/advisories/new), or
- email **security@meno.so** (fallback: **mac@meno.so**).

Include a description, affected version(s), and a reproduction if possible. We aim to
acknowledge reports within a few business days and will keep you updated on the fix.

## Supported versions

Fixes land in the latest published `meno-astro` release on npm. Please verify against
the latest version before reporting.
