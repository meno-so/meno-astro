# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment and discrimination are not
tolerated. We want participation in this project to be a positive experience for
everyone.

## Enforcement

Report unacceptable behaviour to **conduct@meno.so** (fallback: **mac@meno.so**). All
reports will be reviewed and handled confidentially.
